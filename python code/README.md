###File overview
File: mpr_visc_theory.py
Core module implementing the MPR viscosity theory: data structures, model evaluation, and helper functions for plotting.

File: example.py
Standalone script that imports mpr_visc_theory, defines input data, runs the model, and prints or plots the results.

Running the example script
example.py is a ready-to-run demo of the module usage. It typically:

1) Imports mpr_visc_theory
2) Defines or loads example data
3) Calls the relevant model functions/classes
4) Prints numerical results and/or produces plots

To run:

python3 example.py
